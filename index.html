Sou estudante de Tecnologia da Informação e estou construindo minha carreira como desenvolvedor front-end, com foco em criar interfaces modernas, responsivas e funcionais. Desde que comecei a estudar programação, me apaixonei pelo processo de transformar uma ideia em algo que funciona de verdade na tela.

Tenho me dedicado a projetos práticos com HTML, CSS, JavaScript, Git e GitHub, sempre buscando entender não só como fazer, mas por que fazer daquela forma. Gosto de resolver problemas e aprender com cada erro no caminho.

02 Projetos

JS conversor-de-moedas

HTML · CSS · JavaScript · API de câmbio

Problema: criar uma ferramenta prática e rápida para conversão entre moedas em tempo real.

Solução: interface responsiva que consome dados de uma API de câmbio, atualizando os valores automaticamente.

Resultado: aplicação funcional e com código organizado, que fortaleceu meu domínio em consumo de APIs.

JS sistema-help-desk

HTML · CSS · JavaScript

Problema: organizar a abertura e o gerenciamento de chamados técnicos de forma prática.

Solução: interface para cadastro e acompanhamento de chamados, com lógica de estruturação de dados.

Resultado: bom exercício de lógica de programação e construção de interfaces funcionais.

HTML super-mario-encanador

HTML · CSS · JavaScript

Problema: criar uma página criativa pra praticar front-end fora do contexto corporativo.

Solução: página temática inspirada no universo do Super Mario, com elementos interativos e responsiva.

Resultado: mais domínio em estilização, responsividade e experiência do usuário.
